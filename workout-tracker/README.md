# Workout Tracker
This is a workout tracker that save workout information by use of MongoDB/Mongoose and Express.

## Description 
This app is used to track single or multiple daily workouts in a given day. The user is able to track the name, typle, weight, set, reps, and duration of exercise. If there was a distance traveled, the user will also be able to log it as well.
